Transparenz

So rechnet dieses Tool

MitgliederSimple ist eine lokale Desktop-Anwendung. Alle Daten bleiben auf Ihrem Rechner — es gibt keine Cloud-Verbindung, keine Telemetrie, keinen Server-Kontakt.

  • Mitgliederdaten werden in einer lokalen SQLite-Datenbank gespeichert
  • Beitragsberechnung erfolgt nach den hinterlegten Beitragsklassen
  • CSV-Export erzeugt Dateien im deutschen Format (Semikolon-getrennt, UTF-8)
  • Keine automatischen Updates — Sie entscheiden, wann Sie aktualisieren
